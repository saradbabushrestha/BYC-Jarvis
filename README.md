# 🤖 BYC Jarvis

> *"I am Jarvis. I run on vibes, an API key, and one slightly oversized `knowledge.js` file."* 😎

## What is this? 🤔

Honestly? **Not much serious tech here.** Don't @ me. 🙅

This is just a little **show-and-tell demo** to make people in a room go *"ooooh AI"* 😲 at the BYC Conference (Reshaping Municipal Governance in the Age of AI — fancy, right?).

It's basically:
- An Express server 🚂 that does the bare minimum
- A `knowledge.js` file that is *suspiciously* large 📚 (we don't talk about it)
- The Anthropic SDK doing 99% of the actual heavy lifting 🦾
- Me, taking 100% of the credit 🏆

## The "Tech Stack" 🛠️ (using the term loosely)

- Node + Express → so it feels legit
- `@anthropic-ai/sdk` → the part that's actually smart
- A `.env` file with one (1) precious API key 🔑✨
- Vibes → load-bearing

## How to run it 🏃

```bash
npm install        # download half the internet
cp .env.example .env   # then paste your secret sauce 🔑
npm start          # cross your fingers 🤞
```

If it works on the first try, that's the *real* AI miracle, not the chatbot. 🎉

## Disclaimer ⚠️

No robots were harmed. No municipalities were actually governed. This is purely to demonstrate **"hey look, AI can do a thing!"** and nothing more. 😂

If you came here expecting groundbreaking engineering... 😬 awkward. Enjoy the show instead. 🍿
